***ОПИСАНИЕ***

***На языке программирования Си разработана программа graph.c для построения графиков функций, задаваемых произвольными выражениями. Для вычисления значений выражений используется алгоритм Дейкстры для перевода выражения в польскую нотацию. Сборка программы осуществляется при помощи makefile. Исполняемый файл хранится в папке build с именем graph. Некорректными считаются выражения, содержащие любые ошибки, к ним относятся: несбалансированные или пустые скобки, а также их некорректная вложенность, наличие недопустимых символов, функций или операторов, любые отклонения от правил построения математических выражений. Если выражение невозможно преобразовать в корректную математическую форму для построения графика, выводится ERROR.***

* ***Условия***
  
Размер поля для вывода графиков - прямоугольник 25 на 80 точек.

Область определения - [0; 4 Пи]

Область значений - [-1; 1]

Ориентация координатной плоскости - ось ординат направлена вверх, ось абсцисс - направо. 

Центр координат - середина левой границы (точка {0, 12}). 

Значения округляются по правилам математики.

На вход принимаются целые числа, неизвестное 'x', операторы '+', '-', '*', '/' '^', скобки, функции "sin(...)", "cos(...)", "tan(...)", "ctg(...)", "ln(...)", "sqrt(...)".

<br/>

* ***Графика***
  
График выводится с помощью знаков «Звездочек» для линии графика и « . » для всего остального пространства экрана.

* ***Модули***

graph.c - главный модуль

my_io - модуль для ввода произвольного выражения;

check - модуль для проверки выражения на правильность;

spaces - модуль для удаления лишних пробелов и добавлении пробелов между токенами;

list_of_tokens - модуль для получения списка токенов;

my_stack - модуль для стэка;

polish_notation - модуль для перевода в польскую нотацию и вычисления по ней;

matrix - модуль для отрисовки графика функции.
